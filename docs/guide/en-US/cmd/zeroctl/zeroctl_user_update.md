## zeroctl user update

Update a user resource

### Synopsis

Update a user resource.

 Can only update nickname, email and phone.

 NOTICE: field will be updated to zero value if not specified.

```
zeroctl user update USERNAME
```

### Examples

```
  # Update use foo's information
  zeroctl user update --nickname=foo2 --email=foo@qq.com --phone=1812883xxxx
```
